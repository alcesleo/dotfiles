# Dotfiles

## Instructions

1. Set zsh as the default shell. `chsh -s zsh`
2. Install [Prezto](https://github.com/sorin-ionescu/prezto)
3. Backup your previous configuration-files, these scripts sure as hell won't.
4. Run `source dottify.sh
